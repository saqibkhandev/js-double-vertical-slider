# JS Double Vertical Slider ⬆️⬇️

A vertical double-slider demo using HTML, CSS, and JavaScript. Demonstrates layout and synchronized sliding panels.

## Features
- Dual-panel vertical slider
- Simple controls and animations

## Technologies Used
- HTML
- CSS
- JavaScript

## Project Structure
```
index.html — demo
styles.css — layout and animations
script.js — slider controls
```

## Installation / How to run
1. Clone repo.
2. Open `index.html` in a browser.
